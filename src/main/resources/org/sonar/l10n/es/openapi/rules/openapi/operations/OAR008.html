Los verbos permitidos por el estándar son: POST, GET, PUT, PATCH y DELETE. El resto de verbos aunque esten permitidos por REST no deben usarse.

Ejemplo de código no compatible (OpenAPI 2)

swagger: "2.0"
info:
  title: Swagger Petstore
  version: "1.0"
paths:
  /pets:
    post:
      responses: 
        200:
          description: OK
    get:
      responses: 
        200:
          description: OK
    put:
      responses: 
        200:
          description: OK
    patch:
      responses: 
        200:
          description: OK
    delete:
      responses: 
        200:
          description: OK
    head:
      responses: 
        200:
          description: OK
