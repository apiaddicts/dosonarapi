Cada etiqueta debe documentarse con una breve declaración que explique su intención.

Ejemplo de código no compatible (OpenAPI 2)

swagger: "2.0"
info:
  version: 1.0.0
  title: Swagger Petstore
tags:
- name: unused-tag
- name: some-tag
paths:
  /pets:
    get:
      tags:
      - some-tag
      responses:
        200:
          description: OK
