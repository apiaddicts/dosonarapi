A role with forbidden characters may cause problems in some applications.

Noncompliant Code Example (OpenAPI 2)

swagger: "2.0"
x-wso2-security:
  apim:
    x-wso2-scopes:
    - name: read
      roles: R€AD_R*LE
      key: read
      description: Allows users to read data

Compliant Solution (OpenAPI 2)

swagger: "2.0"
x-wso2-security:
  apim:
    x-wso2-scopes:
    - name: read
      roles: READ_ROLE
      key: read
      description: Allows users to read data
